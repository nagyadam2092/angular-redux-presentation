NgRx Effects - our setup

  • dispatch an action (`LOAD_FLOW`)
  • an Effect listens to that action
  • this effect calls a service (`this.shopService.getFlowByPageId`)
  • with RxJS magic (operators) the effect listens to success/failure
  • and shoots another action with data / error message (`(response: Flow) => new shopActions.LoadFlowSuccessAction(response)`)
